This is the client side code of my full stack project named Book-A-Movie which is given to us as Capston project by Almabetter.
In this project i have made a react based webapp where you can book movies which will be stored in database and the recent booked movie details will also be shown there.

This project is live on : https://bookamovie-byrishabh.netlify.app

To run this project in local machine.

Clone the repository, and run npm install. This will download all the dev dependencies required.
Now run npm start. This will start the Client at "http://localhost:3000".

## How to use this WebApp
1-Select Movie name.
2-Select Time slot.
3-Select No. of tickets of required rows.
4-Now click book now. Your Movie will be booked.
To know the last booking,see last booking section.If no movies are booked It will show No Booking found.If Booked it will show last booking details.
